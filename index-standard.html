// About

// Mission

デフエンジニアの会は、ろう者・難聴者のエンジニアを中心に手話や文字で交流し共に学び成長できる環境を作り、デフエンジニアの社会的地位を高めることを目指しています。

// Activities

勉強会・デフハッカソン・交流会などを開催し、デフエンジニア同士のスキルアップ・情報交換の場を設けています。また、オープンソースプロジェクトへの貢献や手話でエンジニア用語を作成し広める活動を通じてデフエンジニアの存在を可視化し、働く環境を良くすることも目指しています。
